# Idee-kwaliteit

Een goed idee:

- Is **specifiek** voor deze reeks (clubs, speeldagen, stand, vorm) — niet generiek “voetbalnieuws”.
- Heeft minstens één **hard feit** met Neon-evidence.
- Opent een **menselijk of spanningspunt** (strijd om titel/degradatie, opvallende scorers, trainerwissel-effect, jeugd die doorbreekt).
- Is **onderscheidend** t.o.v. de andere vier ideeën in dezelfde batch.
- Heeft drie titelvoorstellen die verschillen in hoek (feit / conflict / mens), niet drie parafrases.

Vermijd:

- Clickbait zonder data
- Ideeën die alleen op geruchten rusten
- Dubbele ideeën met dezelfde kernclaim
- Engelse formuleringen
